## 🚀 Getting Started

### Quick Start

1. **Clone the repository:**
```bash
git clone https://github.com/nikhil230145/Portfolio.git
cd Portfolio
```

2. **Start a local server:**
```bash
# Python 3
python -m http.server 8000

# Node.js
npx http-server
```

3. **Open in browser:**
Navigate to `http://localhost:8000`

### No Setup Required
Simply open `index.html` in your browser!

## 🎨 Customization

### Update Your Information

1. Open `index.html`
2. Replace placeholder content:
   - Your name and title
   - About section
   - Projects
   - Skills
   - Contact information

### Change Colors

Edit `css/styles.css`:
```css
:root {
  --primary-color: #0066cc;
  --secondary-color: #00875a;
  /* ... other colors ... */
}
```

### Add Your Projects

Add more project cards in the "Featured Projects" section with your own:
- Project title
- Description
- Technologies used
- Links to live demos or repositories

### Update Skills

Modify skill names, categories, and percentages in the "Skills" section.
