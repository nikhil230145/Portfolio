What is the difference between supervised and unsupervised learning?
ML
Supervised learning trains on labeled data — the model learns a mapping from inputs to known outputs (e.g., predicting house prices, classifying medical images). Unsupervised learning finds hidden patterns in unlabeled data (e.g., clustering customer segments). My carotid artery project uses supervised learning — U-Net trained on labeled medical scans to segment plaque regions.
Explain how U-Net architecture works for image segmentation.
CV
U-Net has an encoder-decoder structure with skip connections. The encoder (contracting path) downsamples feature maps using convolution + max pooling — capturing context. The decoder (expanding path) upsamples and concatenates skip connections from the encoder — recovering spatial resolution. This design preserves fine-grained detail, making it ideal for medical image segmentation where precise boundaries matter.
What is overfitting and how do you prevent it?
ML
Overfitting is when a model memorizes training data but fails to generalize to new data (high training accuracy, low validation accuracy). Prevention techniques: Dropout (randomly disabling neurons), Regularization (L1/L2 penalties), Data augmentation (flipping, rotating images), Early stopping (halt training when val loss increases), and using a validation split. I applied these in my carotid artery project via hyperparameter tuning.
What is the difference between INNER JOIN and LEFT JOIN in SQL?
SQL
INNER JOIN returns only rows where a match exists in both tables. LEFT JOIN returns all rows from the left table and matched rows from the right — unmatched right rows appear as NULL. Use INNER JOIN when you need exact matches; LEFT JOIN when you want to preserve all records from the primary table even without a matching record elsewhere.

What are Python lists vs NumPy arrays — when to use each?
Python
Python lists are dynamic, can hold mixed types, and are great for general-purpose collections. NumPy arrays are homogeneous (one type), memory-efficient, and support vectorized operations — making them 10–100× faster for numerical computation. For ML and data analytics, always prefer NumPy for mathematical operations and Pandas (built on NumPy) for tabular data manipulation.
How do you handle missing data in a dataset?
Data
First identify missing values using df.isnull().sum(). Then choose a strategy: Drop rows/columns if missingness is extensive and data is abundant. Impute with mean/median (numeric), or mode (categorical). Use forward/backward fill for time series. For ML, consider using models like KNNImputer or letting the model handle NaNs (XGBoost supports this). Always document your strategy and validate it doesn't introduce bias.
